# Разработка и Тестирование C# Avalonia приложения


## О проекте

В этом репозитории представлен проект кафе для тестирования.

## Содержимое

- `bug-reports` — файл с баг-репортами

- `CafeProject` — файлы проекта, предоставлена версия с ошибками для тестирования


## Что тестируется

- Авторизация пользователей с разделением прав доступа
- CRU операции с сотрудниками, заказами, сменами
- Генерация отчетов с выбором формата (PDF, Excel)

## Используемые технологии

- C# Avalonia

- Ручное тестирование

## Артефакты тестирования

- Баг-репорты
